MANUAL DE LENGUAJE DE MARCADO HTML

Este documento es un manual para aprender lo más básico de html, que significa Lenguaje de Marcas de HiperTexto.

Un lenguaje de marcado es una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen información adicional acerca de la estructura del texto o su presentación. Las etiquetas son lo que iría dentro del marbete. Pero Nosotros les llamamos directamente etiquetas.

Podemos subdividir las marcas según su función:

Marcado de presentación:

Indica el formato del texto, es decir cómo ha de presentarse el documento. Da la estructua del texto. Añade códigos para que sepamos lo que son las cosas. Con una compración podemos decir que sería como decir el título, autor, editorial. Estas etiquetas suelen estar ocultas al usuario.

Marcado de pocedimiento:

También está enfocado hacia la presentación del texto, sin embargo, las marcas que formatean el texto son visibles para el usuario y permiten procesamiento (le indican al ordenador cómo hacer las cosas con un conjunto de acciones) según el tipo de etiqueta.

Marcado semántico: Utiliza las marcas o etiquetas para describir los fragmentos de texto.

Cuando nació este lenguaje sólo tenía 28 marcas. Pero tuvo mucho éxito desde el inicio. Luego evolucionó metiendo muchas más etiquetas.

Todas las etiquetas que tienen contenido se abren y se cierran. Por ejemplo: <p>Toda etiqueta que tiene contenido lleva otra de cierre la cual lleva el slash después del signo menor que</p>

Las etiquetas llamadas vacías las cuales no encierran ningún texto, no llevan etiqueta de cierre. Por ejemplo: <img>, <br>, <hr>...

Al inicio del documento, dentro del elemento "head" se declara el documento así: <!DOCTYPEhtml>
Se declara el idioma predeterminado del texto de la página web con el atributo de idioma de este modo: <html lang="es">. Para declarar la lengua polaca escribimos "pl", la francesa "fr", la inglesa "en".

METADATOS

Es información que envía el servidort al cliente. Define una propiedad en un elemento y le da valor a esa propiedad.
Su estructura es la siguiente: <elemento atributo="valor">
El metadato<lang> especifica el idioma base de los valores de los atributos y del texto contenido en un elemento. Su sintaxis es la siguiente: lang="Valor que especifica el idioma".

    El atributo lang es útil para:
  • Ayudar a los motores de búsqueda.
  • Ayudar a los sintetizadores de voz.
  • Ayudar al agente de usuario a hacer decisiones sobre separación de palabras, ligaduras, y espaciado.
  • Ayudar a los verificadores de ortografía y gramática.
Para declarar un idioma concreto ponemos el valor que html asigna a cada idioma. Para el español "es", el Polaco "pl", el francés "fr", el inglés "en" y así con todos los idiomas.
.- El metadato <charset> establece el tipo de carácter de este modo:charset=""utf-8". "utf-8" significa el uso de los caracteres de lengua española en la página web.
.- Un atributo de la etiqueta meta es content , el cual es utilizado para dar valores a los metadatos de la página.
Establece el tipo de carácter así <name="Autor" content="Ignacio">.

El uso de texto para comentar las imágenes es una regla fundamental que debemos observar en la creación de sitios web por varias razones:
Los invidentes no pueden interpretar una imagen sin su respectivo comentario a ésta. Para ésto existen navegadores textuales para invidentes.
El uso de comentarios resuelve el problema y permite optimizar la página web. La sintaxis correcta de los comentarios es:
El signo menor que, el signo admiración hacia abajo y dos guiones, debajo el comentario y debajo dos guiones con el signo mayor que.

CREAR UNA TABLA

Para crear una tabla tenemos los siguientes elmentos <table border=número>, <captiongt>, <thead>, <tr>, <td>, <tbody>. <thead> es como el head pero para una tabla. <caption> es como un tituar encima de una tabla o de una imagen. Ha de estar dentro del thead.
Definimos los estilos de la tabla en el <style> dentro del <head> del documento. Para crear el marco de la tabla hemos de escribir en el <head> dentro de <style> table border: número de pixeles del marco y su color, y para las líneas interiores: td {border: número de pixeles y color del marco}.
A la tabla que hemos puesto como ejmplo le hemos aplicado: un borde de un pixel <table border= 1px> y para las líneas de dentro td {border: 2px solid black;}.
<thead> es como el head pero para una tabla. <tr>(del inglés table row) son las filas de la tabla y con td (del ingles table data cell) añadimos los datos de las columnas.
